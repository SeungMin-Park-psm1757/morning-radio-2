# Starter Overlay

This folder contains an additive starter layout for the weekly quantum feature.

It is intentionally conservative:
- new modules live under `src/morning_radio/weekly_quantum/`
- tests live under `tests/weekly_quantum/`
- workflow is separate from the current daily workflow

This is not presented as production-ready code.
It is a scaffold that gives Codex clearer anchors:
- module names
- data model shapes
- CLI entrypoints
- test boundaries
