# Category Briefs Prompt Template

You are preparing a Korean weekly quantum-technology briefing.

Input stories are already deduplicated representatives.
For each category, produce compact JSON-ready brief items with:
- headline
- summary
- why_it_matters
- confidence

Rules:
- Keep it calm, factual, and radio-friendly.
- Avoid repeating the same event.
- Prefer concrete developments over vague hype.
- Note when a story is commentary or analysis rather than straight news.
- If evidence is thin, lower confidence rather than inventing detail.
