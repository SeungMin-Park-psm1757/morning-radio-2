# Install Guide

## Recommended usage

1. Open your existing `daily-news-summary` repository locally.
2. Copy the contents of this pack into the repository root.
3. Commit the docs/skills/templates on a feature branch.
4. Launch Codex from the repository root.
5. Start with one of the prompts in:
   - `docs/quantum_weekly_radio/CODEX_STARTER_COMMANDS_KO.md`

## Why copy into repo root?

Codex discovers:
- repo-level `AGENTS.md`
- repo-level `.agents/skills/*`

So these files should live inside the repository you want Codex to work on.
